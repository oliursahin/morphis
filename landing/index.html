about

morphis is a lightweight, privacy-first email client for Gmail. all your data stays on your machine. built with Rust and Tauri for a native experience with a tiny footprint.

no cloud servers, no analytics, no tracking. your emails are stored in a local SQLite database and never sent anywhere except Gmail's own API. keyboard-driven, fast, and designed to get out of your way.

navigation

Keyboard first. navigate and manage your entire inbox from the keyboard. triage, reply, archive, search — all without reaching for the mouse. vim-inspired motions that feel natural from day one.

J / K to move between threads. Enter to open. E to archive. C to compose. / to search. Cmd+K for the command palette.

organization

Inbox splits. automatically categorize incoming emails into focused tabs — Important, Calendar, GitHub, Others. each split has its own thread count so you always know what needs attention.

customizable queries let you define your own splits. set up per-account configurations that match your workflow.

search

Full-text search. powered by Tantivy, the same engine behind Quickwit. search across all your emails instantly — subject, body, sender, labels. everything is indexed locally on your machine.

sync

Background sync. continuous sync with Gmail keeps your inbox up to date. new emails appear instantly. the sync engine runs in the background — no manual refresh, no stale data.

accounts

Multiple accounts. manage several Gmail accounts from a single window. switch between personal and work inboxes instantly. each account gets its own split configuration, sync state, and search index.

privacy

SQLite database. nothing leaves your device. emails are cached locally for instant access and full-text search. the database lives in your home directory — back it up, move it, delete it.

Zero telemetry. no analytics, no crash reporting, no usage tracking. morphis doesn't phone home.

Open source. MIT licensed. read the code, audit it, fork it.

stack

Rust backend via Tauri for native performance and a small binary. SolidJS frontend for a reactive, snappy UI. Tantivy for full-text search. SQLite for local persistence.

requires Rust 1.77.2+ and Node.js 18+. macOS only (for now).
